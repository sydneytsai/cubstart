JavaScript Playground

Project description

This project is a virtual playground, where users can interact with the various functions programmed onto the website. The functions include a counting button, a todo list maker, and a button that generates activities for when the user is looking to pass time.

How I made it

This project was made using JavaScript, HTML, and CSS. The purpose was to gain a better understanding of the foundations of JavaScript projects and manipulating the DOM.
